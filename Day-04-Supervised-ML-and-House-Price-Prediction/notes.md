# 🚀 Day 4 - Supervised Machine Learning Revision & House Price Prediction Assignment

## 📘 Supervised Machine Learning Revision

Today I revised important concepts of Supervised Machine Learning.

Supervised Learning is a type of Machine Learning where models learn using labeled data.

The model learns patterns from input features and predicts the target output.

### Common Types:
- Regression
- Classification

---

## 📌 Regression

Regression is used when the target variable is continuous.

Examples:
- House price prediction
- Salary prediction
- Temperature prediction

---

## 📌 Classification

Classification is used when the target variable belongs to categories.

Examples:
- Spam detection
- Disease prediction
- Fraud detection

---

# 🏠 Assignment Project - House Price Prediction

## 📘 Problem Statement

A real estate company named **HomeVista Properties** wants to automate its house pricing process using Machine Learning.

The goal is to build a regression model that can predict house prices automatically using historical housing data.

As a Machine Learning Engineer, the task is to:
- Analyze the dataset
- Perform data preprocessing
- Train a Linear Regression model
- Evaluate model performance

---
