# 🛒 Assignment Worked On: ShopSmart E-Commerce Problem

A real-world machine learning assignment focused on predicting whether a visitor would complete a purchase on an e-commerce platform.

The dataset contained **12,330 user sessions** collected over one year and included:

* Page Visits
* Time Spent on Website Sections
* Bounce Rate
* Exit Rate
* Traffic Source
* User Interaction Behaviour
* Numerical and Categorical Features

# 📊 Key Learnings

* Decision Tree Classifier and Regressor solve different types of problems
* Variance reduction is the core idea behind regression trees
* Pre-pruning and post-pruning help reduce overfitting
* Real-world datasets require preprocessing and imbalance handling
* F1-Score is often more informative than accuracy on imbalanced datasets
* Tree-based models are highly interpretable and easy to explain

---

# 💡 Key Insight

Decision Trees are easy to understand and visualize, but without pruning and proper tuning they can overfit by memorizing training data instead of learning meaningful patterns.

Pruning helps strike the right balance between model complexity and generalization.
