# 📊 Key Learnings

- SVM works by finding the optimal hyperplane that maximizes the margin between classes.
- Support vectors are the most important data points that influence the decision boundary.
- Larger margins generally improve model generalization.
- Non-linear problems can be solved using kernel functions.
- SVR applies the same margin-maximization principle to regression tasks.
- Understanding the intuition behind SVM makes the mathematical concepts much easier to grasp.

---

# 💡 Key Insight

SVM is a powerful algorithm because it focuses on the most critical data points rather than the entire dataset.

A strong conceptual understanding of support vectors, margins, and hyperplanes helps in selecting appropriate kernels and tuning model parameters for different machine learning problems.
