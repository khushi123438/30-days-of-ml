## 🔍 Key Concepts

### Anomaly Detection
Anomaly Detection is the process of identifying rare observations that deviate from the majority of data.

### Types of Anomalies
- **Point Anomaly** – Single unusual data point.
- **Contextual Anomaly** – Unusual in a specific context.
- **Collective Anomaly** – A group of observations behaving abnormally.

### DBSCAN
- Density-based clustering algorithm.
- Labels sparse points as outliers.
- Effective for arbitrary-shaped clusters.

### Isolation Forest
- Detects anomalies by isolating observations using random splits.
- Fast and scalable for large datasets.

### Local Outlier Factor (LOF)
- Detects anomalies based on local density differences.
- Useful when anomaly scores depend on neighborhood density.

## 🎯 Key Takeaways

- Anomaly Detection helps identify rare and unusual events.
- DBSCAN treats noise points as anomalies.
- Isolation Forest isolates anomalies efficiently using random partitions.
- LOF detects anomalies based on local density variations.
- Different anomaly detection techniques perform better depending on the dataset characteristics.
