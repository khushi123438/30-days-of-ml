# 🌳 Day 10 Highlights

## 📘 CreditWise Loan Approval System (Problem Understanding & ML Design)

Today I started working on a real-world machine learning project focused on **loan approval prediction for SecureTrust Bank**.

Unlike previous days, the focus was on **problem understanding and ML pipeline design instead of model building**.

---

### 🏦 Problem Statement

SecureTrust Bank processes loan applications manually, which leads to:

- ❌ Slow decision-making  
- ❌ Inconsistent approvals  
- ❌ Bias in evaluation  

---

### 🎯 Objective

To build a Machine Learning system that can:

- Analyze applicant financial data  
- Learn patterns from historical data  
- Predict loan approval (Approved / Rejected)  
- Improve speed and fairness  

---

### 📊 Dataset Overview

- Applicant Income & Co-applicant Income  
- Employment Status, Age, Gender  
- Credit Score & Existing Loans  
- Debt-to-Income Ratio (DTI)  
- Savings & Collateral Value  
- Loan Amount & Loan Term  
- Loan Purpose, Property Area, Education Level  
- Target: Loan_Approved (1/0)  

---

### ⚙️ Tasks Performed

- Understood business problem  
- Converted problem into binary classification  
- Identified important features  
- Studied dataset structure  
- Planned preprocessing steps  
- Designed ML pipeline  

---

### 💡 Key Understanding

Real-world ML starts with **problem understanding, not coding**.  
A strong ML pipeline begins with clear business logic.
