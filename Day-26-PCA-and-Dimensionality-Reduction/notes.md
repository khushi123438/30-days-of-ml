# 🚀 Day 26 - Principal Component Analysis (PCA) & Dimensionality Reduction

Today, I explored **Dimensionality Reduction Techniques** and learned about **Principal Component Analysis (PCA)**, one of the most powerful techniques used to reduce the number of features in a dataset while preserving the most important information.

As datasets grow in size and complexity, high dimensionality can lead to increased computational cost, overfitting, and visualization challenges. PCA provides an efficient way to transform data into a lower-dimensional space while retaining maximum variance.

Through theoretical concepts and practical implementation, I gained a deeper understanding of how dimensionality reduction improves machine learning workflows.

# 📌 Key Learnings

## 🔹 Understanding the Curse of Dimensionality

Learned why machine learning models often struggle with datasets containing a large number of features.

* Increased computational cost
* Higher risk of overfitting
* Reduced model efficiency
* Difficult data visualization

## 🔹 Feature Selection vs Feature Extraction

Explored two approaches for reducing dimensions.

### Feature Selection

* Chooses important existing features
* Preserves original feature meaning
* Simpler interpretation

### Feature Extraction

* Creates new features from existing ones
* Captures more information efficiently
* Used by PCA

## 🔹 Understanding PCA

PCA transforms data into a new coordinate system where:

* First component captures maximum variance
* Second component captures remaining variance
* Components remain independent

This helps compress information into fewer dimensions.

## 🔹 Explained Variance

Learned how Explained Variance Ratio helps determine how much information each principal component retains.

* Higher variance means more information
* Helps select optimal components
* Balances dimensionality reduction and information preservation

## 🔹 Practical Implementation of PCA

Implemented PCA using Python and observed how high-dimensional datasets can be represented using fewer dimensions.

The process involved:

1. Standardizing data
2. Applying PCA transformation
3. Selecting principal components
4. Visualizing reduced-dimensional data

This demonstrated how PCA simplifies datasets without significant information loss.
