# 🧠 Core Concepts Studied

## 📌 Support Vector Machines (SVM)

SVM is a supervised machine learning algorithm used for both classification and regression tasks. It works by finding the optimal hyperplane that best separates different classes.

---

## 🔹 Hyperplane

- A decision boundary used to separate classes
- SVM aims to find the hyperplane with the maximum margin

---

## 🔹 Support Vectors

- Data points closest to the decision boundary
- These points influence the position of the hyperplane

---

## 🔹 Margin

- Distance between the hyperplane and support vectors
- Larger margins generally improve generalization

---

## 🔹 Kernel Trick

- Allows SVM to solve non-linear problems
- Transforms data into higher-dimensional space without explicitly computing coordinates

---

## 🔹 Common Kernel Types

### Linear Kernel

- Used when data is linearly separable
- Fast and computationally efficient

### Polynomial Kernel

- Captures polynomial relationships between features
- Useful when data is not perfectly linear

### RBF (Radial Basis Function) Kernel

- Most commonly used kernel
- Handles complex non-linear decision boundaries effectively

---

## 🌲 Ensemble Learning

Ensemble Learning combines multiple machine learning models to produce better predictions than a single model.

---

## 🔹 Bagging

- Trains multiple models independently
- Reduces variance
- Improves model stability

### Example:
- Random Forest

---

## 🔹 Random Forest

- Collection of multiple Decision Trees
- Each tree learns from a random subset of data
- Final prediction is based on majority voting or averaging

---

## 🔹 Boosting

- Models are trained sequentially
- Each new model focuses on correcting previous mistakes
- Improves overall prediction performance

---

# 📊 Key Learnings

- SVM finds the optimal decision boundary using support vectors
- Larger margins improve model generalization
- Kernel functions enable non-linear classification
- Hyperparameter tuning greatly affects model performance
- Ensemble methods improve accuracy and robustness
- Random Forest and Boosting are among the most powerful ensemble techniques

---

# 💡 Key Insight

A single model can perform well, but combining multiple models intelligently often leads to better, more stable, and more robust predictions.

This principle forms the foundation of Ensemble Learning.
