# 🚀 Day 7 - Scratch Implementation of Machine Learning Algorithms

---

## 📘 Problem Statement

Most Machine Learning models are usually used through libraries like Scikit-learn, but it is very important to understand how these algorithms actually work internally.

The goal of today’s task was to implement core ML algorithms from scratch to deeply understand their mathematical logic and prediction process.

---

# 🎯 Objectives

- Implement Machine Learning algorithms without using libraries
- Understand internal working of regression and classification models
- Build intuition behind predictions and decision-making
- Compare different algorithm behaviors

---

# 🧠 Algorithms Implemented

- Linear Regression (from scratch)
- Logistic Regression (from scratch)
- K-Nearest Neighbours (KNN) Classifier
- KNN Regressor

---

# ⚙️ Machine Learning Workflow

## ✅ Linear Regression

- Implemented using mathematical best-fit line concept
- Understood slope and intercept role in prediction
- Built intuition of continuous value prediction

---

## ✅ Logistic Regression

- Implemented sigmoid function manually
- Converted linear output into probability values
- Understood decision boundary formation

---

## ✅ KNN Classifier

- Implemented Euclidean distance calculation
- Classified based on nearest neighbors majority voting
- Understood distance-based learning approach

---

## ✅ KNN Regressor

- Extended KNN for regression problems
- Used average of nearest neighbors for prediction
- Compared classification vs regression behavior

---

# 📊 Key Concepts Learned

- Gradient-based intuition in regression models
- Sigmoid function for probability mapping
- Distance metrics in KNN
- Difference between classification and regression models
- Importance of understanding algorithms beyond libraries

---

# 💡 Key Insight

Building models from scratch gives a deep understanding of how machine learning actually works. Once the logic is clear, using libraries becomes much more meaningful and powerful.
