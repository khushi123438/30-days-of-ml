# 🚀 Day 5 - Employee Turnover Prediction using Logistic Regression

## 📘 Problem Statement

A multinational company TalentCore Pvt. Ltd. was facing increasing employee resignations, leading to higher recruitment costs and project delays.

The goal was to build a Machine Learning model that can predict whether an employee is likely to leave the company based on various employee-related factors.

---

# 🎯 Objectives

- Build a baseline Logistic Regression model
- Improve performance using:
  - L1 Regularization
  - L2 Regularization
- Compare model performances
- Recommend the best model

### 📌 L2 Regularization (Ridge)
