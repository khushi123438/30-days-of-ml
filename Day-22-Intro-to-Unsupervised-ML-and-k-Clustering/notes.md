# 📌 Key Learnings

## 🔹 Understanding Unsupervised Learning

Unsupervised Learning allows machines to identify patterns and relationships in data without any labeled outputs. The model independently discovers structures hidden within datasets.

## 🔹 What is Clustering?

Clustering is the process of grouping similar data points together while keeping dissimilar points in separate groups. The goal is to maximize similarity within clusters and minimize similarity between different clusters.

## 🔹 How K-Means Works

The K-Means algorithm follows these steps:

1. Choose the number of clusters (**K**).
2. Randomly initialize **K centroids**.
3. Assign each data point to its nearest centroid.
4. Compute new centroids based on assigned points.
5. Repeat until centroids stop changing significantly.

This iterative process helps create meaningful clusters from unlabeled data.

## 🔹 Importance of Choosing K

The performance of K-Means heavily depends on selecting the correct value of **K**.

* Too few clusters may merge distinct groups.
* Too many clusters can over-segment the data.

## 🔹 Elbow Method

The Elbow Method helps determine the optimal value of **K** by plotting WCSS against different K values.

* Lower WCSS indicates tighter clusters.
* As K increases, WCSS decreases.
* The "elbow point" in the graph represents the best balance between cluster quality and complexity.
